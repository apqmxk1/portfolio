# Simple Portfolio Website

This is a simple two-page portfolio website made with HTML, CSS, and basic JavaScript.

## Files

- `index.html`: self-introduction page
- `activities.html`: activity list page
- `styles.css`: modern responsive Material-style design
- `script.js`: basic button interaction

## How to Open

Open `index.html` in your browser.

Then use the top navigation to move between:

- About
- Activities

## Customization

Replace the sample text with your own name, email, skills, goals, and activities.
